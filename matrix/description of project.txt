* This project is based on topic - Matrix Operation

In this we  calculate matrix operation  like- addition,subtraction,multiplication and transpose of matrices.

*Function used :

1.Add(): user can calculate the sum of any number of matrices(what ever user wish)  of any size (m*n).

2. subtract(): user can calculate the difference of any number of matrices (what ever user wish) of any size (m*n). 

3. mult() : user can calculate the product of two matrices of any size (m*n). 

4.Transpose() : converting row into column.
